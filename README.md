# Yan.Pro — Платформа вызова такси

Полнофункциональная платформа вызова такси (аналог Яндекс Go / Uber), состоящая из четырёх компонентов:

| Компонент | Описание |
|---|---|
| **Android Driver App** | Нативное Android-приложение для водителей (Kotlin) |
| **Passenger PWA** | Прогрессивное веб-приложение для пассажиров (HTML/CSS/JS) |
| **Node.js Server** | Socket.IO сервер — брокер связи между водителями и пассажирами |
| **Admin Dashboard** | Панель мониторинга статуса сервера в реальном времени |

## Возможности

### Водитель (Android)
- Карта с местоположением пассажиров и маршрутом
- Переключение онлайн/офлайн статуса
- Приём заявок от пассажиров с принятием и отказом
- Чат с каждым пассажиром
- Интеграция с Яндекс Навигатором (диплинк `yandexnavi://build_route_on_map`)
- Режим Picture-in-Picture во время поездки
- Поиск адресов с автодополнением и историей

### Пассажир (PWA)
- Определение текущего местоположения
- Поиск адреса назначения
- Отслеживание позиции водителя на карте
- Чат с водителем
- Отмена поездки

### Сервер
- Маршрутизация событий в реальном времени (Socket.IO)
- API для админки и health-check
- Статическая раздача PWA и дашборда

## Стек технологий

**Android:** Kotlin, MapLibre SDK, Socket.IO Client, Coroutines, ViewModel + StateFlow, MVVM

**Сервер:** Node.js, Express.js, Socket.IO

**PWA:** Vanilla JS, MapLibre GL JS, Socket.IO Client, Nominatim Geocoding

**Карти:** OpenFreeMap (открытые тайлы)

## Запуск

### Сервер

```bash
cd server
npm install
npm start
```

Сервер запускается на порту `3002`. После запуска доступны:
- Пассажирское приложение: `http://localhost:3002/passenger`
- Админ-панель: `http://localhost:3002/admin`
- Health-check: `http://localhost:3002/health`

### Android-приложение

```bash
./gradlew assembleDebug
# или
./gradlew installDebug
```

По умолчанию приложение подключается к `http://127.0.0.1:3002`. Адрес сервера можно изменить через значок шестерёнки в приложении. Для работы на нескольких устройствах укажите LAN IP сервера (например, `http://192.168.1.42:3002`).

### Требования
- JDK 17
- Android SDK (compileSdk 34)
- Node.js

## Архитектура

- Состояние хранится в оперативной памяти сервера (сбрасывается при перезапуске)
- Водитель видит всех ожидающих пассажиров, но обслуживает одного за раз
- Все пользовательские строки на русском языке
